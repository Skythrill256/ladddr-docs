## Props

| Prop               | Type              | Default | Description                                                  |
| ------------------ | ----------------- | ------- | ------------------------------------------------------------ |
| `onBalanceChange`  | `(conversions: Record<'ETH' \| 'Gwei' \| 'Wei', string>) => void` | | Optional callback function receiving converted balance values |
| `className`        | `string`          |         | Optional CSS class for additional styling                    |

## Validation Schemas

The component uses Zod schemas for input validation:

```tsx
const ethSchema = z.string().regex(/^\d+(\.\d+)?$/, "Invalid ETH format");
const gweiSchema = z.string().regex(/^\d+(\.\d+)?$/, "Invalid Gwei format");
const weiSchema = z.string().regex(/^\d+$/, "Invalid Wei format");
```

## Error Handling

- Validates input for each Ethereum unit (ETH, Gwei, Wei)
- Displays error messages for incorrect input formats
- Prevents invalid conversions
